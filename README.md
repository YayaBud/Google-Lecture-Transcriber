# NoteFlow

**AI-Powered Note Taking for Students & Professionals.**

NoteFlow is a modern, high-performance web application that transforms live lectures into structured, perfect notes instantly. Built with a focus on speed, beautiful UI, and seamless user experience.

![NoteFlow Dashboard Preview](https://github.com/user-attachments/assets/placeholder)

## Features

- **Live Transcription UI**: Real-time visualization of speech-to-text processing.
- **AI Summarization**: Automatically extracts key concepts and action items from lectures.
- **Modern Aesthetic**: Clean, Apple-inspired UI with "Bento grid" layouts and glassmorphism.
- **Dark/Light Mode**: Fully responsive theme toggling with smooth animations.
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.
- **Lightning Fast**: Built with Vite and optimized for performance.

## Tech Stack

- **Frontend Framework**: React + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router DOM
- **State Management**: React Hooks & Context

## Installation
